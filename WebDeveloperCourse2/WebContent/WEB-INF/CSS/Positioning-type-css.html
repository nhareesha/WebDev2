Positioning elements

Positioning of elements is of 3 types

  1. relative : Is relative to the element other wise where it would be
  2. absolute : using this takes the element completely off the flow and aligns it with respect tot he top left corner of the page
  3. fixed:Same as absolute but when page is scrolling it stays to be in the fixed position .Eg : Menu bar moving along with page
